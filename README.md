# HyperLocal Service Marketplace

A community-based platform that connects customers with trusted local service providers for home maintenance and repair.

## Tech Stack

### Frontend
- React.js
- Vite
- Tailwind CSS
- Axios

### Backend
- Java 17
- Spring Boot
- Spring Security
- Spring Data JPA

### Database
- MySQL

## Features

- User Authentication
- Service Booking
- Worker Assignment
- Admin Dashboard
- QR Code Payment
- Ratings & Reviews
